var标识符

该关键字定义的变量是作用域内的一个局部标量。也就是说，如果在函数内声明一个变量，该变量在函数退出后会被销毁。

function test(){
var message = "hi";//局部变量
}
test();
alert(message);//error!

不过可以省略var直接给变量赋值，不过此时没有定义，这样在外边调用一次函数，该变量就有了定义，可以在函数外任何地方使用（虽然省略var操作符可以定义全局变量，但是不推荐这样用，首先它在严格模式下会报错，而且，这样做也会由于相应的变量不会立马就有定义而导致不必要的混乱 ）

function test(){
   message = "hi";  //全局变量
}
test();
alert(message);    //"hi"
